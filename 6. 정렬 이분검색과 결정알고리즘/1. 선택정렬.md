# 1. 선택정렬

Status: 6. Sorting and Searching(정렬 이분검색과 결정알고리즘)

설명

N개이 숫자가 입력되면 오름차순으로 정렬하여 출력하는 프로그램을 작성하세요.

정렬하는 방법은 선택정렬입니다.

입력

첫 번째 줄에 자연수 N(1<=N<=100)이 주어집니다.

두 번째 줄에 N개의 자연수가 공백을 사이에 두고 입력됩니다. 각 자연수는 정수형 범위 안에 있습니다.

출력

오름차순으로 정렬된 수열을 출력합니다.

예시 입력 1

```
6
13 5 11 7 23 15

```

예시 출력 1

```
5 7 11 13 15 23
```

- 정답
    
    ```java
    package test;
    
    import java.util.ArrayList;
    import java.util.Scanner;
    
    public class Main {
    
    	public ArrayList<Integer> solution(int n, int[] arr) {
    		ArrayList<Integer> answer = new ArrayList<>();
    
    		// 13 5 11 5 23 15
    		for (int i = 0; i < n - 1; i++) { // n-1 안해도 되지만 최소한으로 돌리기~
    			int idx = i;
    
    			// 5 11 7 23 15
    			for (int j = i + 1; j < n; j++) {
    
    				// 둘 다 제일 처음 for문 시작한다 가졍하여
    				// arr[0+1] < arr[0] -> 5 < 13
    				if (arr[j] < arr[idx]) {
    					idx = j; // idx = 1
    				}
    				int tmp = arr[i]; // tmp = 13
    				arr[i] = arr[idx]; // arr[0] = 5
    				arr[idx] = tmp; // arr[1] = 13
    
    			}
    			answer.add(arr[i]);
    		}
    
    		return answer;
    	}
    
    	public static void main(String[] args) {
    		Main main = new Main();
    		Scanner scan = new Scanner(System.in);
    
    		int n = scan.nextInt();
    
    		int[] arr = new int[n];
    		for (int i = 0; i < n; i++) {
    			arr[i] = scan.nextInt();
    		}
    
    		for (int x : main.solution(n, arr))
    			System.out.print(x + " ");
    
    	}
    }
    ```
